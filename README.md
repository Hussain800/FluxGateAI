# FLUXGATE: NEXUS Command Center

FLUXGATE is an advanced Traffic & Logistics Command Center designed to optimize school transportation and traffic flow. It provides real-time visualization, predictive analytics, and automated logic for managing student pickup/dropoff protocols, bus coordination, and congestion control.

## 🚀 Key Features

- **Digital Twin Dashboard**: Real-time 3D/2D visualization of school zones, traffic gates, and bus bays.
- **Predictive Analytics**: AI-driven arrival prediction and "Golden Minute" efficiency tracking.
- **Congestion Control**: Automated triggers for gate control and surge alerts based on traffic density.
- **Protocol Management**: Automated workflows for Student/Parent authentication and vehicle verification (RFID/NOL).
- **Incident Logging**: Live feed of traffic incidents and system health monitoring.
- **Companion App Integration**: Seamless synchronization with the mobile app for parents and staff.

## 🛠 Tech Stack

- **Framework**: React 18, Vite
- **Language**: TypeScript
- **Styling**: Tailwind CSS, Shadcn/UI
- **Animation**: Framer Motion
- **State Management**: React Query, Context API
- **Visualization**: Recharts, Lucide React
